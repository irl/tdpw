Flat Description Language

The Flat Description Language is a declarative programming language for defining objects in a 2D universe. It is built of a series of operators with zero or more operands. All operands for a given operator are compulsary, and are delimited by spaces. Each new operator must be on a new line.

Shape Type

circles
This operator takes no operands, and sets the type of shape to circle. If squares has previously been set, this will be overwritten.
squares
This operator takes no operands, and sets the type of shape to square. If circles has previously bee n set, this will be overwritten.

Collision Type

Between Shapes

elastic
This operator takes no operands, and sets the type of collision to elastic. If inelastic has previously been set, this will be overwritten.
inelastic
This operator takes no operands, and sets the type of collision to inelastic. If elastic has previously been set, this will be overwritten.
nocollide
This operator takes no operands, and disables collisions between shapes. This operator will overwrite elastic and inelastic and can also be overwritten by them.

Walls

wrap
This operator takes no operands, and removes collisions with walls. Shapes that would otherwise have had an elastic collision with the wall will wrap around to the wall on the opposite side. If nowrap has previously been set, this will be overwritten.
nowrap
This operator takes no operands, and sets shapes to have elastic collisions with the walls. If wrap has previously been set, this will be overwritten.

Animation Options

frames
This operator takes one operand, an integer representing the number of frames to be generated. This will overwrite any previous definition set by frames.
freeze
This operator takes one operand, an integer representing the number of frames to have no movement at the start of the animation. These frames count towards the total number of frames set by the frames operator, and will not add extra frames to the animation. This will overwrite any previous definition set by freeze.

Shape Definition

shape
This operator takes the following arguments to define a shape:
  • X coordinate of top-left corner (integer)
  • Y coordinate of top-left corner (integer)
  • Width of shape (integer)
  • Initial velocity of X component vector (integer)
  • Initial velocity of Y component vector (integer)
  • Colour (red|green|blue|yellow|cyan|purple|white|random)
Subsequent definitions will create new shapes up to a maximum of 1000 shapes.

Testing

echo
This command takes one operand, a string to be printed back. This is used for testing the interactive shell, and serves no other useful purpose.

Control

run
This command takes no operands, and will begin creating the animation.
exit
This command takes no operands, and will exit without creating the animation.
