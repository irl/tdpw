Usage

Programming

Batch mode

To run with a pre-prepared script, use input redirection at the shell, for example:

./a.out < scripts/single.tdpw

Interactive mode

Launch the program with ./a.out and you will be presented with a prompt. Here you can issue FDL operators and then to create the animation with the command run.

Output

Two output files will be produced: anim.gif and anim.mpg in the same directory you ran the application in.
